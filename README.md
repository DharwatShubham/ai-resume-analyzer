# AI Resume Analyzer

An AI-powered Resume Analyzer built using Java, Spring Boot, and Google Gemini AI. The application analyzes uploaded resumes, calculates an ATS score, identifies strengths and weaknesses, and provides AI-generated suggestions to improve resume quality.

## Features

- Upload PDF Resume
- ATS Score Calculation
- AI Resume Review using Gemini AI
- Resume Strengths & Weaknesses
- ATS Improvement Suggestions
- Recommended Job Role
- Resume Score Dashboard

## Technologies Used

- Java
- Spring Boot
- Thymeleaf
- HTML
- CSS
- Bootstrap
- Google Gemini API
- Maven

## Installation

1. Clone the repository

```bash
git clone https://github.com/DharwatShubham/ai-resume-analyzer.git
```

2. Open the project in IntelliJ IDEA or Eclipse.

3. Set the Gemini API Key as an environment variable.

```
GEMINI_API_KEY=YOUR_API_KEY
```

4. Run

```bash
mvn spring-boot:run
```

5. Open

```
http://localhost:8080
```
